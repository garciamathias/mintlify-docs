## Core Features

### Image Generation
Generate photorealistic lifestyle scenes from simple product photos in seconds.

- **AI-Powered Backgrounds**: Automatically creates contextual environments
- **Material Consistency**: Maintains accurate textures and finishes
- **Realistic Shadows**: Proper lighting and shadow generation
- **Accurate Scaling**: Intelligent size relationships between objects

## Editing Tools

### Magic Tools

<AccordionGroup>
  <Accordion title="Magic Erase" icon="eraser">
    Paint over any object to remove it. AI seamlessly recreates the background.
  </Accordion>
  <Accordion title="Magic Replace" icon="wand-sparkles">
    Select an element and type what should appear instead (e.g., "oak legs" to "metal legs").
  </Accordion>
</AccordionGroup>

### Adjustment Tools

<AccordionGroup>
  <Accordion title="Resize Product" icon="expand">
    Drag to scale the product. Defines floor level and final perspective.
  </Accordion>
  <Accordion title="Change Aspect Ratio" icon="crop">
    Switch between square, portrait, or landscape without quality loss.
  </Accordion>
  <Accordion title="Relight" icon="sun">
    Post-generation lighting adjustment with Low, Medium, High presets (4 variants each).
  </Accordion>
</AccordionGroup>
