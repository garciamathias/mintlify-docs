## Workflow Optimization

- [ ] Product properly positioned on canvas
- [ ] Adequate space left for scene generation
- [ ] All accessories added (if needed)
- [ ] Prompt reviewed and specific
- [ ] Reference image uploaded (if using)

## Common Challenges & Solutions

### Preventing Issues

<AccordionGroup>
  <Accordion title="Multi-Product Perspective Mismatch">
    **Prevention**: Shoot all products from identical angles
    **Fix**: Re-photograph to match perspectives
  </Accordion>
  <Accordion title="Inconsistent Lighting">
    **Prevention**: Use Relight tool consistently
    **Fix**: Select lighting intensity that matches scene
  </Accordion>
  <Accordion title="Scale Problems">
    **Prevention**: Reference real-world dimensions
    **Fix**: Use Resize Product tool before generating
  </Accordion>
  <Accordion title="Background Clutter">
    **Prevention**: Add "minimal accessories" to prompt
    **Fix**: Use Magic Erase on unwanted elements
  </Accordion>
</AccordionGroup>

### Quick Fix Guide

| Issue | Symptom | Fast Fix |
|-------|---------|----------|
| **Perspective** | Objects clash visually | Use photos from same angle |
| **Lighting** | Product too bright/dark | Adjust Relight intensity |
| **Scale** | Items look wrong size | Manually resize products |
| **Clutter** | Busy scene | Add "clean space" to prompt |
