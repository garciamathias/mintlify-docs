## Batch Processing Overview

Batch mode enables you to generate lifestyle scenes for multiple products simultaneously, saving hours of manual work.

<Note>
  **Available on**: Team and Enterprise plans only
</Note>

## How Batch Mode Works

<Steps>
  <Step title="Prepare Your Products">
    Organize product images and ensure consistent quality
  </Step>
  <Step title="Create Template">
    Design your scene template with prompts and settings
  </Step>
  <Step title="Upload Batch">
    Import via CSV or select multiple products
  </Step>
  <Step title="Process">
    AI generates all scenes automatically
  </Step>
  <Step title="Review & Download">
    Check results and download approved images
  </Step>
</Steps>

## Input Methods

### CSV Upload

Structure your CSV with these columns:

| Column | Required | Description |
|--------|----------|-------------|
| **sku** | Yes | Unique product identifier |
| **image_url** | Yes | Direct link to product image |
| **product_name** | No | For organization |
| **prompt_override** | No | Custom prompt for specific product |
| **category** | No | For grouping results |

<CodeGroup>
```csv Example CSV
sku,image_url,product_name,prompt_override,category
SOFA-001,https://example.com/sofa1.jpg,Modern Gray Sofa,,Living Room
CHAIR-002,https://example.com/chair2.jpg,Velvet Accent Chair,"Luxury bedroom with soft lighting",Bedroom
TABLE-003,https://example.com/table3.jpg,Oak Coffee Table,,Living Room
```
</CodeGroup>

### Batch Controls

- **Parallel Processing**: Up to 10 simultaneous generations
- **Queue Management**: Pause, resume, or cancel batches
- **Priority Ordering**: Process urgent items first
- **Error Handling**: Auto-retry failed generations

## Best Practices

<AccordionGroup>
  <Accordion title="Prepare products consistently">
    - Use same photography angle
    - Maintain consistent lighting
    - Standardize image dimensions
    - Clean, high-res source files
  </Accordion>
  <Accordion title="Test before full batch">
    - Run 5-10 product sample
    - Refine prompts based on results
    - Adjust settings as needed
    - Save successful template
  </Accordion>
  <Accordion title="Organize efficiently">
    - Group by product category
    - Use clear naming conventions
    - Tag for easy filtering
    - Create approval workflows
  </Accordion>
  <Accordion title="Monitor progress">
    - Check first few results
    - Adjust if needed early
    - Set up notifications
    - Plan review sessions
  </Accordion>
</AccordionGroup>

## API Integration

<Note>
  **Enterprise only**: Integrate batch processing into your workflow via API.
</Note>

```python
# Example API batch submission
import presti_api

batch = presti_api.create_batch(
    template_id="living_room_modern",
    products=[
        {"sku": "SOFA-001", "image_url": "..."},
        {"sku": "SOFA-002", "image_url": "..."}
    ],
    settings={
        "resolution": "4K",
        "lighting": "medium"
    }
)
```
