## Post-Generation Editing

After generating your scene, use Presti.ai's suite of editing tools to perfect every detail without starting over.

<Info>
  **No Additional Cost**: All editing tools are free to use. Credits only apply to final downloads.
</Info>

## Magic Tools

### Magic Erase

Remove unwanted elements seamlessly from your scene.

<Tabs>
  <Tab title="How to Use">
    1. Select the Magic Erase tool
    2. Paint over objects to remove
    3. AI fills the area naturally
    4. Maintains scene consistency
  </Tab>
  <Tab title="Best For">
    - Removing clutter
    - Cleaning up backgrounds
    - Eliminating distractions
    - Simplifying compositions
  </Tab>
  <Tab title="Pro Tips">
    - Paint slightly beyond object edges
    - Work on one object at a time
    - Use for small to medium elements
    - Great for post-generation cleanup
  </Tab>
</Tabs>

### Magic Replace

Transform elements while preserving scene composition.

<Steps>
  <Step title="Select Element">
    Click on the object you want to change
  </Step>
  <Step title="Describe Replacement">
    Type what should appear instead (e.g., "metal legs" to replace "wooden legs")
  </Step>
  <Step title="Generate">
    AI seamlessly swaps the element while maintaining lighting and perspective
  </Step>
</Steps>

**Common Replacements:**
- Material changes (wood to metal, fabric to leather)
- Color variations (beige to gray, oak to walnut)
- Style updates (modern to vintage, simple to ornate)

## Adjustment Tools

### Resize Product

Adjust product scale and positioning after generation.

<Warning>
  **Important**: This tool is especially useful for fixing unwanted elements like extra legs appearing under furniture corners.
</Warning>

**How to Fix Common Issues:**
1. Open the edit panel
2. Select "Resize Product"
3. Adjust the bounding box to exclude unwanted parts
4. Validate to regenerate without artifacts

### Change Aspect Ratio

Switch between different canvas formats without regenerating.

| Format | Ratio | Best For |
|--------|-------|----------|
| **Square** | 1:1 | Instagram, thumbnails |
| **Portrait** | 3:4 | Mobile, Pinterest |
| **Landscape** | 4:3 | Web banners, desktop |

<Note>
  AI intelligently extends or crops the scene to fit the new ratio while preserving your product placement.
</Note>

## Enhancement Tools

### Accessorize

Add decorative elements to enhance your scene.

<Tabs>
  <Tab title="Process">
    1. **Before generating background** - this is crucial
    2. Select Accessorize tool
    3. Brush where accessories should appear
    4. Describe each accessory precisely
    5. Generate scene with accessories included
  </Tab>
  <Tab title="Brush Tips">
    - Brush size hints at accessory scale
    - Make brushed area slightly larger than intended
    - Be specific in descriptions
    - Add multiple accessories in one go
  </Tab>
  <Tab title="Examples">
    **Good Descriptions:**
    - "Potted monstera plant in white ceramic planter"
    - "Stack of hardcover books with neutral spines"
    - "Geometric gold table lamp with white shade"
    
    **Poor Descriptions:**
    - "Plant"
    - "Books"
    - "Lamp"
  </Tab>
</Tabs>

### Replace Product

Swap your main product while keeping the generated scene.

**Perfect for:**
- Testing different products in same setting
- Creating product family shots
- A/B testing variations
- Maintaining scene consistency across catalog

<Info>
  The scene lighting, shadows, and composition automatically adjust to the new product.
</Info>

## Advanced Techniques

### Multi-Tool Workflows

Combine tools for complex edits:

1. **Generate** base scene
2. **Magic Erase** unwanted elements
3. **Accessorize** to add details
4. **Relight** for perfect mood
5. **Magic Replace** for final touches

### Batch Editing

On Team plans, apply edits across multiple images:
- Consistent lighting adjustments
- Standardized aspect ratios
- Bulk accessory additions
- Series replacements
