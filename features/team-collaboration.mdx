## Review Mode

### Collaborative Feedback

Enable stakeholders to provide contextual feedback directly on images.

<Steps>
  <Step title="Share for Review">
    Send review links to stakeholders - no account needed
  </Step>
  <Step title="Gather Feedback">
    Reviewers click on images to leave positioned comments
  </Step>
  <Step title="Track Status">
    Monitor approval progress with status indicators
  </Step>
  <Step title="Implement Changes">
    Address feedback and update status in real-time
  </Step>
</Steps>

### Review Tools

| Feature | Description | Use Case |

  </Accordion>

  </Accordion>

    - View shared albums
    - Leave comments
    - Approve/reject
    - Download approved
    - No editing access
  </Accordion>
  <Accordion title="Guest" icon="user">
    - Limited time access
    - Specific album only
    - Comment privileges
    - No downloads
    - Auto-expiring links
  </Accordion>
